#  깜빡이 💡

### 2025 멋쟁이사자처럼 대학 13기 4호선톤 - '집중력 방범대' 팀

**숨겨진 집중력의 불을 켜주는 아동교육 서비스**

> 기간 : 25. 10. 12 ~ 25. 11. 14.

## Introduce

1️⃣ 깜빡이'는 만 5세~10세 아동의 집중력 향상을 목표로 하는 아동 교육 서비스 </br>
2️⃣ 서비스는 연구 논문 기반의 게임인 '뿅뿅 아기별'과 '꼬마 교통지킴이'를 통해 집중력과 기억력, 문제해결력을 훈련 </br>
3️⃣ 부모님을 위한 게임 결과를 분석하는 AI 기반 집중력 분석 레포트는 아이의 학습 능력을 정량적으로 진단하여 보호자에게 제공

## 문제인식
- 대한민국 아동의 집중력 저하 문제 강조
- 만 5세부터 14세 사이의 ADHD(주의력 결핍 과잉행동장애) 환자 수는 2017년 3만 6,960명에서 2020년 4만 6,098명으로 꾸준히 증가하는 추세
- 주의력 결핍은 아동의 인지, 정서, 행동에 심각한 문제발생
-> 약 50% 정도는 청소년기까지 지속되는 장기적인 문제 심화

## 게임 기능 설명

### 1. 뿅뿅 아기별

> [!NOTE]
> 뿅뿅 아기별은 아이가 우주 탐험대 대원이 되어, 길을 잃은 '아기 별들'이 깜빡인 순서를 기억하여 원래의 '별자리'로 돌려보내는 임무를 수행하는 게임

#### 1) 게임 목표
- 이 게임은 초등 저학년에게 권장하며, 무작위로 깜빡이는 별의 위치와 순서를 기억하고 입력하는
  훈련을 통해 집중력과 기억력을 한 번에 향상시키는 것

#### 2) 게임 규칙 및 점수 기준
- 인지 단계: 랜덤한 위치의 별들이 정해진 순서대로 깜빡이며 등장
- 기억/입력 단계: 플레이어는 깜빡인 순서와 위치를 정확하게 기억하여 별을 클릭하고 입력
- 정해진 시간 제한 내에 입력을 완료하지 못할 경우 게임이 종료되며 정확한 입력 시 점수(1점)를 획득
- 아이의 회복탄력성 및 좌절감 방지를 위해 감점은 제외

### 2. 꼬마 교통지킴이

> [!NOTE]
> '꼬마 교통지킴이'는 아이가 안전한 도시를 만드는 '신호등 지킴이'가 되어, 화면 속의 자동차를 안전하게 지나갈 수 있도록 돕는 임무를 수행하는 게임

#### 1) 게임 목표
- 미취학 아동 후반(만 5세 ~ 7세) 하며, 랜덤으로 변하는 물체에 집중하여 빠르게 반응하며, 절제하는 능력을 훈련시키는 것을 목표

#### 2) 게임 규칙 & 점수기준
- 게임은 신호등의 색깔 변화에 맞춰 정확한 시점에 달려 또는 멈춰를 클릭하여 점수를 획득하는 방식으로 진행 
- 누적 3번 잘못 클릭했을 때 게임이 종료되며, 정확하게 반응(클릭)할 때마다 1점을 획득

### 3. AI 기반 레포트'AI 기반 집중력 분석 레포트'

> [!NOTE]
> 아이의 게임 결과를 종합하여 학습 역량을 정량적으로 분석하고 아이의 학습 결과를 보호자가 직접 확인할 수 있도록 제공하는 핵심 기능
> 이 레포트는 두 가지 핵심 게임인 '뿅뿅 아기별'과 '꼬마 교통지킴이'의 플레이 데이터를 기반으로 다음 세 가지 주요 영역을 분석한다. 

#### 1) 아이의 집중력 지수
- 아이의 전반적인 학습 능력을 정량적으로 측정
  
#### 2) 충동 조절 능력
- 아이가 행동을 억제하고 적절한 시점에 반응하는 능력을 측정, 게임의 과학적 근거인 Go/No-Go 과제를 통해 측정되는 충동성과 반응 억제 조절 능력을 반영
- 활용 데이터: '꼬마 교통지킴이' 게임의 최대 라운드 도달률 / 평균 도달 라운드, 반응속도를 바탕으로 분석
  
#### 3) 주의력 능력
- 분석 목표: 아이가 장시간 미션에 집중하고 오류 없이 정보를 처리하는 능력을 파악, 이는 N-Back 효과와 지속성을 평가하는 데 활용
- 활용 데이터: '뿅뿅 아기별' 게임의 최대 라운드 도달률 / 평균 도달 라운드, 오답률, 그리고 제한시간 초과 비율을 바탕으로 분석
